a. What is the Document Object Model (DOM)?

Ans: Document Object Model

b. How do you select an element from the DOM?

1. document.getElementById('id'): Selects an element by its id

2. document.querySelector('selector'): Selects the first element that matches a CSS selector.

3 .document.querySelectorAll('selector'): Selects all elements matching a CSS selector.

c. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events for its child elements.Reduces the number of event listeners by using one for many child elements.

d. How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles in the DOM using: Attributes: setAttribute(name, value): Sets or updates an attribute. getAttribute(name): Retrieves an attribute's value. removeAttribute(name): Removes an attribute.
